# 💫 About Me:
I am an aspiring Data Scientist and MLOps Engineer focused on building end‑to‑end machine learning solutions, from data ingestion to production deployment. I work extensively with time‑series forecasting, NLP/LLMs, RAG pipelines, and deep learning using Python, TensorFlow, PyTorch, Scikit‑learn, and LightGBM.<br><br>Recently, I have delivered projects such as bike rental demand forecasting, Apple stock price prediction, and customer segmentation, where I designed full ML pipelines with robust EDA, feature engineering, model evaluation, and deployment. I enjoy translating complex models into interactive analytics using Streamlit, Dockerized web apps, and BI tools like Tableau and Power BI.<br><br>I’ve also contributed to India’s first Telugu Large Language Model in collaboration with Meta and IIIT Hyderabad, working on transformer‑based systems, large‑scale token processing, and RAG for multilingual NLP. My goal is to build reliable, production‑ready AI systems that deliver clear, measurable business impact while continuously deepening my expertise in modern ML and MLOps.
